## Overview

A classic snake game built with Python. Simple, fun, and addictive!

## How to Play

- Use arrow keys to move the snake
- Eat food to grow longer
- Don't hit the walls or yourself
- Try to get the highest score!

## Features

- Classic gameplay
- Score tracking
- Simple controls
- Game over detection

## Tech Stack

- Python 3.x
- Pygame library
